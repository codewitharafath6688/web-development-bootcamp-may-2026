


# 📚 BookCourier

**Enriching the Virtual Book Library** 🚀 **Live URL:** [https://bookcourier-auth.web.app/](https://bookcourier-auth.web.app/)

---

## 📖 Project Overview
**BookCourier** is a dynamic virtual book library platform designed to bridge the gap between avid readers and their favorite books. The platform curation ranges from timeless classics to contemporary bestsellers, making it effortless for users to discover, purchase, and access popular literature globally. With secure payment integrations and integrated support infrastructure, BookCourier provides a seamless end-to-end reading and purchasing ecosystem.

---

## ✨ Key Features

- **Curated Book Collections:** Explore diverse and high-quality sets of books tailored to various genres and reader preferences.
- **Featured Authors:** Dedicated sections highlighting popular and trending authors alongside their complete catalogs.
- **Frictionless Checkout:** An intuitive user flow designed to make acquiring your next read as easy as a few clicks.
- **Stripe Payment Gateway:** Fully integrated international payment processing using Stripe, enabling secure transactions worldwide.
- **On-Demand Assistance:** Integrated communication and service system via direct calling support to provide immediate solutions for user inquiries.

---

## 🛠️ Tech Stack & Packages

The project leverages modern web development tools and npm packages to ensure speed, security, and scalability:

- **Frontend Core:** React.js
- **Routing:** `react-router-dom` (Declarative routing for smooth multi-page navigation)
- **Styling:** Tailwind CSS (Utility-first CSS framework for highly responsive UI)
- **State Management & Data Fetching:** `@tanstack/react-query` (Asynchronous state management and caching)
- **HTTP Client:** `axios` (Promise-based data fetching from external endpoints)
- **Maps Integration:** `leaflet` (Open-source interactive maps for location/delivery visualization)
- **Authentication & Hosting:** Firebase (Secure user authentication and fast hosting infrastructure)
- **UI Feedback:** `sweetalert2` / `sweetalert` (Beautiful, responsive, and customizable popup alerts)
